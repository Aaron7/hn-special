HN Special

An extension for Hacker News

Fork me on GitHub

HN Special is a Chrome extension which adds a completely new visual style to Hacker News and a few new features, such as:

  • Infinite scrolling
  • Open links in new tabs
  • More accurate domain names (includes subdomains)

You can choose which features you want enabled from the settings menu.

This extension is completely open source, so even more features may be coming in the future if it becomes popular. If you wish to implement a new feature or make some changes, feel free to contribute on GitHub!

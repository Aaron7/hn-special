HN Special

An extension for Hacker News

Fork me on GitHub

HN Special is a Chrome extension which adds a completely new visual style to Hacker News and a few new features, such as:

  • Infinite scrolling
  • Open links in new tabs
  • More accurate domain names

You can choose which features you want enabled from the settings menu.

This extension is completely open source. If you wish to add a new feature or make some changes, feel free to contribute on GitHub!
